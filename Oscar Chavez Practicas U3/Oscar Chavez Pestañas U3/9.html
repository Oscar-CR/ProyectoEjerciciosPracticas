Código

Algo detallado, aquí está la fuente de Jade. La mayoría de las bibliotecas JS todavía piden un marcado similar que casi no existe aunque mejor.

Cóigo HTML (Jade):

div.tv-tabs
    label.tv-tab(for='tv-tab-1') Tab Name!
    label.tv-tab(for='tv-tab-n') for attr equals the radio ids
   
  input.tv-radio#tv-tab-1(name='any-group-name', type='radio', checked)
  div.tv-content Content goes here!
   
  input.tv-radio#tv-tab-n(name='any-group-name', type='radio')
  div.tv-content More goes here!

Còdigo CSS (lápiz óptico) relevante:

.tv-content
    display none
   
  .tv-radio
    display none
   
    &:checked + .tv-content
      display block
