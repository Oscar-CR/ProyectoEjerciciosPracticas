Proyecto de Prácticas

El objetivo de este proyecto es recopilar los ejercicios y prácticas realizadas a lo largo del semestre, en cada una de las fases de este proyecto se fueron utilizando distintas tecnologias, empezando con HTML, seguido de CSS, posteriormente Javascript y PHP, ademas de incorporar frameworks como Bootstrap y Node.js.
